# Personal Finance Dashboard

A modern and interactive **Personal Finance Dashboard** developed using **HTML, CSS, JavaScript**.

This dashboard enables a user to track their income, expenses, savings, etc., in a visually appealing manner.

---

## Features

- Premium Dark Theme UI Design
- Income vs Expense Trend Chart
- Spending by Category Doughnut Chart
- KPI Cards
- Transaction Search Functionality
- Role-based Switching (Admin/Viewer)
- Add Transaction Button
- Savings Health Insights
- Top Expense Category
- Fully Responsive Design

---

## Technologies Used

- HTML5
- CSS3
- JavaScript
